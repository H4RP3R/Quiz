Разработать ПО с графическим пользовательским интерфейсом для прохождения
викторин со следующим функционалом:

1. Загрузка списка вопросов из JSON файла.
2. Начальный (приветственный) экран, показывающий тему викторины, кол-во
вопросов и кнопку для запуска викторины.
3. Показ вопросов из ранее загруженного списка по одному и 4 варианта ответов
к каждому. Каждый вопрос имеет один верный ответ. Варианты вопросов
кликабельные, правильные и не правильные ответы учитываются для последующего
вывода статистики.
4. Финальный экран с результатами викторины.
